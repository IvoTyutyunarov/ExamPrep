# Project Description
ExamPrep is a database-driven platform designed to help students prepare for exams effectively.  
The system allows users to register, log in, and take various tests or quizzes.  
It stores users’ performance data in a SQL Server database and visualizes key metrics—such as average scores and success rates—using Power BI dashboards.  

The main goal of the project is to provide insights into student performance through interactive data visualizations, helping both learners and educators track progress over time.
